resource模块可以对页面元素进行权限控制。
workflow模块可以对页面元素进行流程控制。
dom元素含有data-resource属性则会进行权限控制，data-resource值不在当前用户所拥有的资源列表中将会移除该dom元素。
dom元素含有data-action的属性会进行流程控制，存在data-bpn属性时会校验该属性和data-action的值是否对应，不对应则会移除该dom元素。
调用方法：
    var resrouce=this.resource;
    var workflow=this.workflow;
    resrouce.assess();
    workflow.assess();
